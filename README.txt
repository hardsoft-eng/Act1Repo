Ce projet est une démonstration d'un exemple de programmation C++.
Il s'agit d'une classe appelée Time pour gérer le temps.
Le fichier Time.h contient les membres et les prototypes des méthodes de la classe Time
Le fichier Time.cpp contient l'implémentation des méthodes de la classe Time
Enfin, Le fichier main.cpp contient le programme principal qui teste notre classe Time
